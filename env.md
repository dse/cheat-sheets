# env

run a program in a modified environment

    env [<option> ...] [-] [<name>=<value> ...] [<cmd> [<arg> ...]]
    
## Linux

-   -i, --ignore-environment
-   -0, --null
-   -u, --unset=<name>
-   -C, --chdir=<dir>
-   -S, --split-string=<s>
-   --block-signal[=<sig>]
-   --default-signal[=<sig>]
-   --ignore-signal[=<sig>]
-   --list-signal-handling
-   -v, --debug
-   --help
-   --version

## FreeBSD

-0, -i, -L, -U <user>[/<class>]
-P <altpath>, -S, -u <name>, -v

## POSIX

-i

## OpenBSD

-i
