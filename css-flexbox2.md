## Container (Parent) Styles

``` css
display: flex;
         inline-flex;

flex-flow: row nowrap;
flex-direction:
    row
    column
    row-reverse
    column-reverse
    // specifies
    // MAIN AXIS
flex-wrap:
    nowrap
    wrap
    wrap-reverse

align-content:

justify-content:
```
